# 🚀 SchemaLens

SchemaLens is a powerful, web-based database diagramming tool that instantly visualizes your database schema using a single smart query.

No sign-up required. No setup headaches. Just paste, generate, and explore.

---

## ✨ Features

### ⚡ Instant Schema Import

Run one query to retrieve your database schema as JSON and automatically generate a clean, interactive ER diagram.

### 🤖 AI-Powered SQL Export

Generate DDL scripts in your preferred SQL dialect.
Easily migrate between:

- MySQL
- PostgreSQL
- SQLite
- MariaDB
- SQL Server

### 🎯 Interactive Editing

Modify tables, columns, and relationships directly in the editor.
Design visually. Export confidently.

### 🔄 Easy Sharing

Import and export diagram files for seamless team collaboration.

---

## 🥇 Why SchemaLens?

- Instantly understand complex database structures
- Reduce cognitive load during development
- Improve team discussions with visual schema diagrams
- Speed up database migrations
- No account required

---

## 🚀 How It Works

1. Run the provided schema query on your database
2. Paste the JSON output into SchemaLens
3. Instantly visualize your database structure
4. Edit, customize, and export
